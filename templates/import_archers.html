Format requis

Le fichier CSV peut être au format FFTA (Code adhérent, Nom, Prénom…) ou export club avec colonnes du type N°, Nom, Prénom (une seule colonne), Date de naissance, etc. Séparateur : point-virgule ou virgule (détection automatique). Si une même étiquette apparaît deux fois (ex. deux « Catégorie »), les deux colonnes sont conservées.

Colonnes usuelles (séparées par des points-virgules ou des virgules) :

  • Code adhérent *
  • Civilité
  • Nom *
  • Prénom
  • DDN
  • Nationalité
  • Type licence
  • Saison
  • Discipline(s)
  • Catégorie âge sportif
  • Catégorie âge
  • Code structure
  • Nom structure
  • Nom du responsable légal
  • Prénom du responsable légal
  • Téléphone du responsable légal
  • Mail du responsable légal
  • Nom du responsable légal secondaire
  • Prénom du responsable légal secondaire
  • Téléphone du responsable légal secondaire
  • Mail du responsable légal secondaire

* Champs obligatoires

CSV avec points-virgules. Encodage reconnu : UTF-8 (avec ou sans BOM), Windows-1252 (Western European — Excel « Enregistrer sous CSV »), Latin-1, etc.

💡 Conseil

Assurez-vous que:

  • Le fichier utilise des points-virgules (;) comme séparateur
  • Les codes adhérents sont uniques
  • Les dates de naissance sont au format JJ/MM/AAAA ou DD/MM/YYYY
